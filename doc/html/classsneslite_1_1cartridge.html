Detailed Description

Cartridge memory allocation.

A typical NES cartridge contains two ROM banks. A character ROM which contains a game's tilesets, and a program ROM which contains the game's binary. Some cartridges come with additional RAM, such as battery-backed work RAM (used for saving), or character RAM (used for bank switching tilesets).
